# Layout Ideas

Each section will be broken up into combos, tap-holds, and base positioning.
The best of each can be combined later.

## Symbol Layer

### Base Positioning

- Option 1

    - Keep in mind what symbols will be on the number layer and keep in mind rolls
        - Open and close html tags (<>, </>)
        - Open and close long lua comments (--[[/--]])
        - Operator formations (+=, !=)
        - Arrows (->/=>)
        - Shell variables ("${}")
        - #!
        - ~/ ./, ../
        - [ ] How many of these are common enough that they could be macros?
